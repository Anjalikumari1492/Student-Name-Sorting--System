# Student Name Sorting Program

## Description
This C++ program accepts student names as input and sorts them alphabetically using the Bubble Sort algorithm.

## Features
- Stores up to 100 student names
- Sorts names alphabetically
- Displays sorted names

## Concepts Used
- Arrays
- Strings
- Loops
- Conditional Statements
- Bubble Sort

## Sample Output

Enter number of students: 5
Enter student names:
Riya
Anjali
Priya
Aman
Neha

Students in Alphabetical Order:
